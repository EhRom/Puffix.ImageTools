# Create a release

The release publication is automated with GitHub Actions and GitHub CLI.

A personnal access token is needed.

|Information|Value|
|---|---|
|Name|`PFX_IMAGES_GH_CLI_TOKEN`|
|Expriration|Short duration|
|Repository|[EhRom/Puffix.ImageTool](https://github.com/EhRom/Puffix.ImageTools)|
|Scope|Repository permissions:<ul><li>Actions (*readonly*)</li><li>Contents (*read write*)</li><li>Metadata (mandatory, *readonly*)</li></ul>|

The content of the PAT is saved in a repository secret.
